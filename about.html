This is a site where I can experiment with sound on the web. To start this means getting to know the web audio api. This site, and the corresponding github repository, will be a place where I can show and share what I'm working on -- all html, css, and javascript to get the examples working. Sometimes I will have a related blog post at my "sandbox" --- see blog here -- but sometimes I will just post the experiment. If you'd like to get in touch, please check out chrisstrelioff.ws, where you will find email, social media links and other information.
