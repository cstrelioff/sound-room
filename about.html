This is a site where I can experiment with sound on the web. To start this means getting to know the web audio api. This site, and the corresponding github repository, will be a place where I can show and share what I'm working on -- all html, css, and javascript to get the examples working. Sometimes I will have a related blog post at my "sandbox" --- see blog here -- but sometimes I will just post the experiment. If you'd like to get in touch, please check out chrisstrelioff.ws, where you will find email, social media links and other information.

external links

Do you have more web audio links? Let me know -- I'd love to know and I'll add them to this list.

  • MDN Web Audio API Docs

    Web Audio API docs at the Mozilla Developer Network.

  • O'Reilly Web Audio API Book

    Boris Smus' book -- very nice, and free online!

  • Web Audio Weekly

    "If you are interested in the Web Audio API, the Web MIDI API, or sound on the web in general, you'll love Web Audio Weekly." -- looks really useful, I subscribed!

  • Synthesising Drum Sounds with the Web Audio API

    A blog post by Chris Lowis at Dev.Opera -- "In this article, we’re going to learn more about the API by building a simple drum machine. We’ll learn just enough about the API to play a simple loop constructed from sounds we synthesise ourselves, and we’ll see how to use recordings of real-life sounds too. From these bare bones, you’ll be able to extend the machine yourself, or apply the techniques you’ve learnt to other applications."

    The github repository is here: https://github.com/chrislo/drum_synthesis and a live verion is here: https://chrislo.github.io/drum_synthesis/

  • 'A brief history of synthesis' [ youtube ]

    Six examples of using the Web Audio API, tracing the history of electronic music, by Chris Lowis -- github repository here: https://github.com/chrislo/synth_history

  • tone.js

    A Web Audio framework for making interactive music in the browser -- github repository here: https://github.com/Tonejs/Tone.js

  • pico.js

    Audio processor for the cross-platform -- github repository here: https://github.com/mohayonao/pico.js

  • MidiWriterJS

    JavaScript library providing an API for generating expressive multi-track MIDI files in browser and Node.

  • WebAudioSynth

    Synthesizer application written in JavaScript using Web Audio API -- github repostitory here: https://github.com/aike/webaudiosynth

  • litsynth

    A Web Audio API synth for demoscene written in literate javascript -- github repository here: https://github.com/padenot/litsynth

  • asynth

    Create midi synths out of javascript functions.

  • dx7-synth-js

    A JavaScript implementation of the Yamaha DX7 synthesizer -- see it live here: http://mmontag.github.io/dx7-synth-js/
